# coe216-hw1

Simple web demo for the COE216 homework.

- Task 1: sampled sinusoids and sum plot (with animation and sampling-rate control).
- Task 2: DTMF keypad tone generation with time-domain and FFT plots.

## Steps to Run Locally

Run in the terminal:

```bash
$ git clone https://github.com/ahmetesad/coe216-hw1
$ cd coe216-hw1
$ python3 -m http.server 5173
```

Then visit: http://localhost:5173
